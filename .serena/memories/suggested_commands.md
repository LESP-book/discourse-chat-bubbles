# Suggested Commands

- Install lint dependency: `yarn install`.
- Lint initializer: `npx eslint javascripts/discourse/api-initializers/chat-bubbles.js`.
- Template lint if template/GJS/theme structure changes: `npx ember-template-lint .`.
- Live preview in a local Discourse host when available: `discourse_theme watch .`.
- Fast repo search: `rg <pattern>` and `rg --files`; quote paths with spaces or shell-sensitive characters.
